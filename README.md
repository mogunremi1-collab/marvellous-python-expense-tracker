# 💰 Python Expense Tracker

A simple yet powerful **command-line Expense Tracker** built with **Python** and **SQLite**.  

Add, view, categorize, edit, delete, and summarize your income & expenses. Perfect for beginners learning Python, databases (SQL), and CLI applications!

![Expense Tracker Demo](https://via.placeholder.com/800x450/1E3A8A/ffffff?text=Python+Expense+Tracker+CLI)

## ✨ Features

- ➕ Add **income** and **expense** transactions
- 🗂️ Support for multiple categories (Food, Transport, Rent, Salary, Entertainment, Utilities, Shopping, etc.)
- 📊 View summaries:
  - Total Income
  - Total Expenses
  - Current Balance
  - Monthly breakdown
  - Spending by category
- 📋 View all transactions in a clean, formatted table
- 🔍 Search/filter transactions by date, category, or description
- ✏️ Edit existing transactions
- 🗑️ Delete transactions
- 💾 Data automatically saved in a local **SQLite** database (`expenses.db`)
- 🧹 User-friendly menu-driven interface

## 🛠️ Technologies Used

- Python 3
- SQLite3 (built-in with Python)
- `tabulate` – for beautiful tables in the terminal
- `datetime` – for handling dates
