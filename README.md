# 💰 Python Expense Tracker

A simple yet powerful command-line Expense Tracker built with **Python** and **SQLite**.

![Expense Tracker Screenshot](screenshot.png)

## ✨ Features

- ➕ Add Income and Expense transactions
- 📋 View all transactions in a clean table
- 📊 Show total Income, Expenses, and Balance
- 🗂️ Categorized transactions
- 💾 Data saved automatically in SQLite database
- 🧹 Simple and easy-to-use menu
